# Rahul Kumar Nayak — Portfolio

Welcome to my personal portfolio website. This repository contains a simple, responsive one-page portfolio built with HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

![Profile photo](Rahul%20Kumar%20Nayak.jpg)

Demo
----
Open the `index.html` file in a browser to view the portfolio locally, or deploy it using GitHub Pages for a live site.

About
-----
This portfolio is a lightweight, static website intended to provide a clean and accessible way to present:
- A short bio and profile image
- Skills and interests
- Project highlights and visuals (e.g. circuit images)
- Contact details and links

Features
--------
- Single-page layout (index.html)
- Custom styles in `Rahul.CSS`
- Interactive behaviors implemented in `first.js`
- Included images for profile and project visuals:
  - `Rahul Kumar Nayak.jpg`
  - `electric circuit.png`

Files and Structure
-------------------
- `index.html` — Main HTML file for the portfolio
- `Rahul.CSS` — Stylesheet for layout, typography, and responsiveness
- `first.js` — JavaScript for interactivity and UI enhancements
- `Rahul Kumar Nayak.jpg` — Profile image
- `electric circuit.png` — Example project image / visual

Getting started (local)
-----------------------
1. Clone the repo:
   ```
   git clone https://github.com/RahulNayak2127/Portfolio.git
   cd Portfolio
   ```
2. Open `index.html` in your browser:
   - Double-click the file or
   - Serve it with a local HTTP server:
     ```
     # Python 3
     python -m http.server 8000
     # then visit http://localhost:8000
     ```

Development notes
-----------------
- Styles: edit `Rahul.CSS` to change colors, layout, and responsive breakpoints.
- Scripts: update `first.js` to add behaviors (animations, form handling, or analytics).
- Add new projects by modifying the relevant section in `index.html` and placing assets in the repository.

Deploying (GitHub Pages)
------------------------
1. Push the repo to your GitHub account (it is already here).
2. Go to the repository Settings → Pages and select the `main` branch and `/ (root)` to publish.
3. The site will be available at `https://RahulNayak2127.github.io/Portfolio/` (or similar) after a few minutes.
